How do you keep GitHub in sync with Cloud9?

1) Push to github from cloud9

  • Log in GitHub from Cloud9
  • Add a git remote in the Cloud9 console

    • git init
    git remote add origin git@github.com:C9Support/testPush.git

  • Add files and commit them

    • git add .
    git commit -m "First commit"

  • Push to github

    • git push -u origin master

    2) Update one or more files in Cloud9 IDE workspace

  • Ensure the GitHub repository is kept synchronized

    • git commit -a -m 'Brief description of the update(s) that were done' (replace with an actual description)
    git push

    3) Delete a file from Cloud9 IDE workspace and GitHub

  • Ensure the GitHub repository is kept synchronized
  • Replace FILENAME with the name of the file being deleted

    • git rm FILENAME
    git commit -m 'Brief description'
    git push

    4) Checke the status of Cloud9 IDE workspace

  • Execute the command "git status" in the Cloud9 IDE terminal

    • git status

    5) Confirme GitHub repository is up-to-date

  • Log in to the GitHub account, navigate to the repository, and view the commit logs and current status of the files in the repository to confirm that GitHub is synchronized with Cloud9 IDE

    In what ways is Cloud9 software practical to use for your organization?

    1. For basic support Cloud9 end users can invite new users to a team and they do not need to provide credit card details on signup. Cloud9 does not need to have access to your credit card info.
    2. For educational purposes Cloud9 is a good option because it can be provided for $1 per month for each team that you set up. It doesn’t matter how many members join it. Once they sign up they can use the normal share flow.
    3. Cloud9 is a low cost alternative to keeping onsite servers up and running.
    4. Since Cloud9 puts important data and applications in the cloud IT crashes and natural disasters can’t affect your company.
    5. Cloud9 provides reliable backup, constant availability, and affordable support.
    6. Cloud9’s automated systems can handle most routine support requirements. If complicated issues arise the live support desk can provide quick assistance.
    7. Time consuming and complicated software deployments to end users will no longer be necessary as they can access software directly from the cloud.
    8. Users can create files and allow other users to access them instead of sending multiple copies back and forth.
    9. Users and clients can work on the same files at the same time in a secure location and don’t have to worry about working on the correct version of a document. Multi-user capabilities allow immediate access to a full desktop and specific files.
    10. Leasing is also available to keep your users on the latest edition of their software at a fraction of the cost. Cloud9 will provide your users with anytime anywhere access. They can handle all of the IT, maintenance, backups and updates that your company needs to operate.
    11. With secure Cloud9 resourcing your company can stop worrying about updates, upgrades, or security.
    12. The Cloud9 service provides full desktop functionality in a real-time environment through the Cloud9 user portal.
    13. Cloud9 can provide hosting services to businesses of any size. It can remove all of the support and maintenance operations while providing a full server.
    14. Cloud9’s maintenance and support frees your company up to focus on what’s important.
    15. Cloud 9 can manage the details of keeping your servers secure. You will know that that all servers contain virus protection and automatic, nightly backups.
    16. Cloud9 can provide Microsoft Exchange Email cloud servers that work on computers, tablets, and smartphones. BlackBerrys can also be synced.
    17. Cloud9 can provide Office 365 hosting.
